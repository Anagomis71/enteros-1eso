# Maestro de los Números Enteros

Juego interactivo para practicar operaciones con números enteros pensado para 1.º de ESO.

## Características

- Dos modos de actividad: operaciones con dos números y operaciones con tres números.
- Opción de operadores mixtos en ejercicios de tres números.
- Marcador grande de puntos.
- Botón para reiniciar el marcador.
- Mensajes visuales al acertar y al fallar.

## Uso

1. Abre `index.html` en el navegador.
2. Elige la actividad que prefieras.
3. Resuelve la operación escribiendo el resultado y pulsa "Comprobar".
4. Usa el botón "Reiniciar marcador" para empezar de nuevo.
